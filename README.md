# Getting Started
Follow these steps to set up the project locally

1. Clone the repo
```
git clone https://github.com/GabrielSpelchuk/layout_landing-page.git
cd layout_landing-page
```
2. Install dependencies
```
npm install
# or
yarn install
```
3. Run the project locally
```
npm start
# or
yarn start
```
